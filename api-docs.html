🔑 API DeepSeek V4

Получение ключа

Отправьте команду /apikey боту @PhotoDocumentsBot_bot. Ключ выдаётся один раз. При компрометации отзовите командой /revoke_apikey и получите новый.

Авторизация

Передавайте ключ в заголовке каждого запроса:

Authorization: Bearer sk-xxxxxxxxxxxxxxxxxxxxxxxx

Базовый URL

https://фото-бот.рф

Примеры использования

1. SSE Streaming — POST /chat

Основной эндпоинт для общения с DeepSeek. Ответ приходит через Server-Sent Events (SSE). Аналогично работают /create_branch, /regenerate_assistant, /regenerate_user, /selfplay.

Параметры: user_id, message, model (flash/pro), chat_id, branch_id, personality, mode (chat/reasoner)

cURL

curl -X POST https://фото-бот.рф/chat \
  -H "Authorization: Bearer sk-xxx" \
  -H "Content-Type: application/json" \
  -d '{"user_id":"123","message":"Привет","model":"flash","chat_id":"default"}'

Python

import requests

headers = {"Authorization": "Bearer sk-xxx"}
data = {"user_id": "123", "message": "Привет", "model": "flash", "chat_id": "default"}

resp = requests.post("https://фото-бот.рф/chat", headers=headers, json=data, stream=True)
for line in resp.iter_lines():
    if line and line.startswith(b"data: "):
        print(line.decode())

JavaScript

const resp = await fetch("https://фото-бот.рф/chat", {
  method: "POST",
  headers: { "Authorization": "Bearer sk-xxx", "Content-Type": "application/json" },
  body: JSON.stringify({ user_id: "123", message: "Привет", model: "flash", chat_id: "default" })
});
const reader = resp.body.getReader();
const decoder = new TextDecoder();
while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  console.log(decoder.decode(value));
}

2. Получение данных — GET /get_chat_history

Возвращает все чаты пользователя в формате JSON. Аналогично работают /get_chat_tree, /get_full_response, /get_balance.

Параметры: user_id, model, meta_only (опционально, только заголовки чатов)

Python

import requests

headers = {"Authorization": "Bearer sk-xxx"}
resp = requests.get("https://фото-бот.рф/get_chat_history",
    headers=headers, params={"user_id": "123", "model": "flash"})
data = resp.json()
# data = {"history": {...}, "titles": {...}, "active_branches": {...}}

JavaScript

const resp = await fetch(
  "https://фото-бот.рф/get_chat_history?user_id=123&model=flash",
  { headers: { "Authorization": "Bearer sk-xxx" } }
);
const data = await resp.json();
console.log(data.history);

3. Управление чатами — POST /rename_chat

Переименовывает чат. Аналогично работают /delete_chat, /share_chat, /rename_branch, /delete_branch, /switch_branch, /copy_history.

Параметры: user_id, model, chat_id, title

Python

import requests

headers = {"Authorization": "Bearer sk-xxx"}
resp = requests.post("https://фото-бот.рф/rename_chat",
    headers=headers,
    json={"user_id": "123", "model": "flash", "chat_id": "abc", "title": "Новый чат"})
print(resp.json())  # {"success": true}

JavaScript

const resp = await fetch("https://фото-бот.рф/rename_chat", {
  method: "POST",
  headers: { "Authorization": "Bearer sk-xxx", "Content-Type": "application/json" },
  body: JSON.stringify({ user_id: "123", model: "flash", chat_id: "abc", title: "Новый чат" })
});
console.log(await resp.json());  // {success: true}
